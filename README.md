#Om prosjektet

Dette er et React-prosjekt bygget med Vite, hvor jeg har utviklet en nettside som henter data fra Ticketmaster sitt API. Nettsiden viser ulike typer informasjon om kommende arrangementer, inkludert bilde, by, dato, tid og mer.

##Prosjektet er utviklet med JavaScript, HTML og SCSS, og fokuserer på å gi brukeren en enkel og oversiktlig måte å utforske arrangementer på.

##Teknologier brukt

- React

- Vite

- JavaScript

- HTML

- SCSS

- Ticketmaster API

##Funksjonalitet

- Henter og viser arrangementer direkte fra Ticketmaster API

- Viser relevant informasjon som bilde, sted, dato og tidspunkt

- Strukturerte og responsive komponenter bygget i React
